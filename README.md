# 📚 Book Recommendation Engine

## 📌 Overview
This project recommends books based on user preferences using machine learning.

## 🚀 Features
- Personalized recommendations
- Similarity-based filtering
- Data preprocessing

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn

## ▶️ How to Run
pip install -r requirements.txt
python app.py
